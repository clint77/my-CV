# Clint Pijuan's CV

This repository contains the source for Clint Pijuan's portfolio and CV website, published with GitHub Pages.

## Website Content

- `index.md` contains the CV displayed on the website.
- `_config.yml` configures the GitHub Pages Jekyll theme.

Update professional content in `index.md`. Keep this README focused on maintaining the repository rather than duplicating CV details.

## Publishing

GitHub Pages publishes the site automatically from the `master` branch. After pushing a change, allow a few minutes for the deployment to finish and then check the public site.

## Validation

Before pushing content changes, run:

```sh
git diff --check
git diff -- index.md _config.yml
```

Preview `index.md` as rendered Markdown and verify headings, lists, dates, and external links.
